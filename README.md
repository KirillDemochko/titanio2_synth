
# TiO₂ Nanotube Structure Prediction

ML-пайплайн для предсказания структурных параметров нанотрубок диоксида титана на основе условий электрохимического анодирования. Проект разработан для оптимизации режимов синтеза в рамках научной работы в группе биомиметических материалов.

## Описание задачи

Модель предсказывает 5 ключевых параметров структуры нанотрубок на основе 13 входных параметров синтеза. Это позволяет виртуально подбирать режимы анодирования и отжига для получения целевой морфологии без проведения "вслепую" множества дорогостоящих лабораторных экспериментов.

### Входные параметры (13 признаков):
* `Ethylene_glycol_vol_percent` (доля EG, %)
* `Water_vol_percent` (доля воды, %)
* `NH4F_wt_percent` (концентрация фтора, вес.%)
* `Glycerol_vol_percent` (доля глицерина, %)
* `HF_vol_percent` (концентрация HF, %)
* `Voltage_V` (напряжение, В)
* `Anodization_time_min` (время анодирования, мин)
* `Temperature_anodization_C` (температура анодирования, °C)
* `Annealing_temperature_C` (температура отжига, °C)
* `Annealing_time_min` (время отжига, мин)
* `Substrate_thickness_mm` (толщина подложки, мм)
* `Substrate_area_cm2` (площадь подложки, см²)
* `Annealing_atmosphere` (атмосфера: Air, N2, Ar, O2, None)

### Целевые переменные (5 параметров):
* `Tube_diameter_nm` — диаметр нанотрубок
* `Tube_length_um` — длина нанотрубок
* `Wall_thickness_nm` — толщина стенок
* `Pore_density_pores_per_um2` — плотность пор
* `Anatase_ratio` — доля анатаза (кристаллической фазы)

## Результаты

Сравнение 5 алгоритмов (CatBoost, Random Forest, GPR, MLP, Ridge) на датасете из 500 сэмплов (разделение 80/20) показало, что **Gaussian Process Regressor (GPR)** стабильно превосходит остальные модели по всем целевым метрикам.

**Лучшие метрики (GPR):**


| Параметр | MAE | RMSE | R² |
| :--- | :--- | :--- | :--- |
| Tube_diameter_nm | 4.4064 | 5.7210 | **0.9855** |
| Tube_length_um | 1.0914 | 1.6152 | **0.9615** |
| Wall_thickness_nm | 0.9248 | 1.1714 | **0.9817** |
| Pore_density_pores_per_um2 | 15.7676 | 23.4874 | **0.9726** |
| Anatase_ratio | 0.0198 | 0.0272 | **0.9948** |

**Средний R²: 0.9792**

> *Почему GPR?* Гауссовские процессы показали наилучшую обобщающую способность, вероятно, благодаря точному учету масштабирования признаков (StandardScaler) и способности моделировать сложные нелинейные зависимости с высокой эффективностью на выборках среднего размера (500 сэмплов).

## Установка и Запуск

1. **Клонирование репозитория**
```bash
git clone <repository_url>
cd titanio2_synth
```


2. **Установка зависимостей**

   ```bash
   pip install -r requirements.txt
   ```

3. **Генерация данных** (если отсутствует `dataset.csv`)

   ```bash
   python data_generator.py
   ```

4. **Обучение моделей**

   Запустите скрипты для обучения конкретных моделей:

   ```bash
   python model_catboost.py
   python model_GPR.py
   # ... (аналогично для RF, MLP, Ridge)
   ```

5. **Сравнение результатов**

   ```bash
   python model_compare.py
   ```
   
   *Этот скрипт загрузит сохраненные предсказания и построит сводные таблицы и графики.*
